# Onion News Style Guide

## Headline Formats
- "Area Man [absurd action]"
- "[Authority Figure] Assures Nation [obviously false thing]"
- "Study Finds [obvious thing stated scientifically]"
- "Breaking: [mundane thing treated as crisis]"
- "[Number]% Of [group] [surprising statistic]"

## Tone
- Deadpan. Never acknowledge the joke.
- Write as if this is completely serious journalism.
- The funnier the premise, the more serious the tone.
