# Cinematic Broadcast — Documentary Segment

You are producing a documentary-style broadcast segment about a current event. This is a FILM — a polished broadcast segment with narrative arc, visual pacing, and dramatic structure. Not a dashboard. Not a status page. Not a grid of cards. A produced broadcast.

## First Step

Discover the full component catalog before producing anything. Study what is available — especially broadcast components (LowerThird, Ticker, Banner, Badge), animation components (Animate, Stagger, Presence), data visualization (Chart, Stat, Sparkline, Table), and layout tools (Split, Overlay, Grid). These are your production tools. A real broadcast uses lower thirds, data graphics, tickers, and dramatic reveals — and you have all of these.

## Quality Standards

- **Broadcast television quality.** Think BBC documentary, 60 Minutes, or a well-produced YouTube essay. Every frame intentional. Every transition motivated. Every piece of text typographically considered.
- **Continuity between segments.** Segments flow into each other with visual logic. Use Animate to bring elements in with cinematic entrances (slide-in-left for lower thirds, fade-in for title cards, scale-up for dramatic reveals). Use Stagger when multiple elements need to appear in sequence.
- **Rich use of broadcast components.** Use LowerThird for correspondent attribution. Use Ticker for running context. Use Banner for breaking developments. Use Chart and Stat for data visualization moments. Use Badge for source labels. Layer them with Overlay for broadcast depth.
- **No generic AI aesthetic.** No centered text on gradients. No uniform card grids. Each segment has its own visual identity while maintaining broadcast coherence.
- **Pacing matters.** A broadcast that rushes feels cheap. One that lingers feels slow. Each scene breathes — establishes itself, delivers content, then cuts to the next. The rhythm of cuts IS the editing.

## Production

Invent a compelling topic and all supporting details — statistics, quotes, developments — from your imagination. Make it vivid, specific, and believable.

Pick a topic from the seed file (or invent your own), then produce:

**Title Sequence** — A dramatic opening that establishes the story's weight. Use Animate to reveal the title with cinematic timing. Include a visual element that signals what the story is about. Hold long enough for impact.

**Investigative Segments** — Each segment gets a distinct visual treatment. A data-driven segment uses Chart, Stat, and Sparkline to let numbers reveal the story. A correspondent segment uses LowerThird for attribution and a different composition. Use Stagger to build up information reveals within segments.

**Data Visualization Moment** — At least one segment where the data IS the story. Use Chart (bar, line, area) to show trends. Use Stat for key numbers. Use Counter for dramatic reveals of quantities. Make the audience feel the numbers.

**Breaking Development** — If warranted, interrupt with Banner and a dramatic visual shift. Use Presence to animate alert elements in and out.

**Closing Summary** — Tie everything together. Recap key findings. Use a composition that feels like resolution — visually distinct from the investigative segments.

Throughout, use Ticker for running context. Use LowerThird for every correspondent/source attribution. Use hard cuts between segments and incremental updates within them. The distinction between cuts and evolution is your primary creative tool.
